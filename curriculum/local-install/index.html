From Zero to OpenClaw in 5 Minutes

Get OpenClaw running locally via npm with full Token Factory integration. Ship your first AI agent in under five minutes.

Who It's For
First-time OpenClaw users who want a working agent fast
Value
Production-ready setup with zero config
Takeaway
A running local OpenClaw agent connected to Token Factory

Prerequisites

  • Node.js 18+
  • npm
  • Terminal / shell access
  • OpenClaw Token Factory API key

Learning Outcomes

  • Install OpenClaw via npm
  • Configure Token Factory credentials
  • Launch your first agent
  • Understand the local dev workflow

Step-by-Step Guide

1
Install OpenClaw 2 min

Install OpenClaw globally via npm and verify the installation.

npm install -g openclaw
openclaw --version
If you see a version number, you're good to go.
2
Set Token Factory Key 1 min

Export your Token Factory API key as an environment variable. This connects your local agent to GPU inference without needing your own GPU.

export OPENCLAW_TOKEN_FACTORY_KEY=your-key-here
Get your key at the Token Factory dashboard. Free tier available.
3
Launch Your Agent 1 min

Start an agent with a single command. Choose your model — GLM-4-Plus is a great default for general tasks.

openclaw agent start --model glm-4-plus
4
Test It 1 min

Open your browser and interact with your agent through the built-in web interface.

open http://localhost:18789

Send your first message. Your agent is live, connected to Token Factory inference, and ready to work.
